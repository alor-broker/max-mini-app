# User

## Поля

| Поле | Тип | Описание |
|------|-----|----------|
| **user_id** | integer <int64> | Идентификатор пользователя или бота |
| **first_name** | string | Отображаемое имя пользователя или бота |
| **last_name** | string Nullable optional | Отображаемая фамилия пользователя. Для ботов это поле не возвращается |
| **username** | string Nullable | Никнейм бота или уникальное публичное имя пользователя. В случае с пользователем может быть null, если тот недоступен или имя не задано |
| **is_bot** | boolean | `true`, если это бот |
| **last_activity_time** | integer <int64> | Время последней активности пользователя или бота в MAX (Unix-время в миллисекундах). Если пользователь отключил в настройках профиля мессенджера MAX возможность видеть, что он в сети онлайн, поле может не возвращаться |
| **name** | string Nullable | Устаревшее поле, скоро будет удалено |

## Пример объекта

```json
{
  "user_id": 0,
  "first_name": "string",
  "last_name": "string",
  "username": "string",
  "is_bot": true,
  "last_activity_time": 0,
  "name": "string"
}
```
