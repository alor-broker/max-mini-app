# MAX для разработчиков

## О платформе

Подключение к платформе MAX для партнёров и её сервисам — чат-ботам, мини-приложениям, каналам — доступно для юрлиц и ИП, которые являются резидентами РФ.

[Платформа MAX для партнёров](https://business.max.ru/self) позволяет выстраивать удобную коммуникацию с клиентами в мессенджере MAX.

С помощью платформы вы можете:
- Создать [чат-боты](#о-чат-ботах) для настройки сценариев диалогов
- Подключить [мини-приложения](#о-мини-приложениях) для обработки запросов и принятия заказов без перехода на внешние платформы
- [Добавить канал](#о-каналах) своей организации в MAX для публикации новостей
- Подключить сервисы партнёров

Платформа MAX для партнёров пока доступна только в веб-версиях:
- Для компьютера — работать лучше на экранах от 1440 px, ограничений на браузеры нет
- Для мобильных устройств с iOS и Android — поддерживаются экраны от 375 px

## О чат-ботах

Чат-боты экономят время.

С помощью чат-ботов вы можете автоматизировать работу с клиентами, вести бизнес, обучать, продавать и развлекать.

Бот создаётся после [подключения](./02_Подключение_к_платформе/connection.md) к платформе MAX для партнёров, добавления там организации и её верификации. Бот проходит [модерацию](./02_Подключение_к_платформе/connection.md#этап-4-модерация) и после этого становится доступен пользователям через MAX.

### Для быстрого старта

Если у вас нет навыков программирования или нужен быстрый старт, используйте [конструкторы](./03_Чат-боты/02_Бот_без_разработки/create.md#конструктор) сервисов партнёров, которые помогут собрать сценарии для бота без кода, по готовым шаблонам.

### Для получения всех возможностей чат-бота

С навыками разработки вы можете [настраивать чат-боты](./03_Чат-боты/03_Бот_с_разработкой/01_Подготовка_и_настройка/prepare.md), используя все возможности API ботов.

[Создать чат-бот на платформе](./03_Чат-боты/01_Создание_чат-бота/bots-create.md)

## Об интеграции с сервисами партнёров

После добавления и модерации бота на платформе вы можете подключить сервисы партнёров:

- **Конструкторы ботов** — для создания сценариев чат-ботов без разработки
- **CRM-системы** — для автоматизации рутинных операций и работы с клиентской базой: лидами, заявками, историей общения с клиентами
- **Чат-платформы** — для общения с клиентами через сервис партнёров

Для подключения к сервисам партнёров используйте токен бота.

## О мини-приложениях

Мини-приложения — это веб-сайты и программы, которые можно запустить прямо в MAX: например, интернет-магазины или игры.

Мини-приложения снижают нагрузку на команды продаж и поддержки за счёт возможности принятия заказов без перехода на внешние ресурсы.

Чтобы создать мини-приложение, нужны навыки программирования.

Мини-приложения доступны на разных устройствах и проще в разработке, чем мобильные приложения. Они работают на стандартных веб-технологиях — HTML, JavaScript, CSS, а ещё их не нужно дорабатывать под разные платформы и модерировать в сторах.

### Мини-приложения не могут существовать изолированно

Для возможности взаимодействия мини-приложений с API подключите библиотеку MAX Bridge. Это посредник на стороне пользователя, с помощью которого приложение обменивается данными с клиентской частью MAX. Библиотека нужна для отправки и получения событий. Подробнее читайте [в разделе «MAX Bridge»](../04_Мини-приложения/bridge.md).

Чтобы пользователи не замечали переходы из чатов в мини-приложения, рекомендуем адаптировать интерфейсы сервисов под дизайн-систему MAX. Создавайте консистентный и привлекательный визуал [с библиотекой React-компонентов MAX UI](../../MAX_UI/README.md).

### Мини-приложения в MAX должны соответствовать правилам

Перед размещением мини-приложения в MAX изучите [Правила платформы](./06_Интеграция_с_партнёрами/rules.md). Ваш сервис должен соответствовать законам Российской Федерации, а также требованиям к его содержанию и функциональности.

[Подключить мини-приложение](../04_Мини-приложения/introduction.md)

## О каналах

В MAX доступны публичные и приватные каналы.

### Публичные каналы

С помощью публичных каналов вы можете взаимодействовать со своей аудиторией через мессенджер MAX, например публиковать новости и анонсировать акции.

### Приватные каналы

Приватные каналы удобно использовать для коммуникаций на ограниченную аудиторию, например, оповещений и публикации новостей только для сотрудников организации. Количество приватных каналов неограниченно — можно создать несколько, например, для разных отделов или филиалов.

Также вы можете использовать приватные каналы для монетизации. Например, публиковать уникальный контент и выдавать к нему доступ только после оплаты или размещать рекламные посты за деньги.

Создание приватных каналов доступно пока только для пользователей из РФ — аккаунт в мессенджере MAX должен быть зарегистрирован на российский номер телефона.

### Каналы с меткой А+

Если у вас уже есть ресурсы с меткой А+ на других платформах, вы можете создать публичные каналы с такими же никами и названиями в MAX.

Для этого воспользуйтесь ботом [Каналы в MAX](https://max.ru/channel_bot). Можно добавить столько каналов, сколько у вас есть ресурсов с меткой А+ на платформах ВКонтакте, Дзен, Одноклассники или в Telegram.

Обратите внимание, что создавать канал в MAX нужно с того же номера, с которого вы подавали заявку на получение метки А+ для стороннего ресурса.

[Создать канал с названием и ником, как у ресурса с меткой А+](../05_Каналы/create.md#как-создать-канал-с-таким-же-названием-как-у-ресурса-с-меткой-А+)

### Канал с выбранным названием

Также вы можете создать публичный канал с выбранным вами названием и автогенерируемым ником.

Создать такой канал можно независимо от наличия ресурсов с меткой А+ и регистрации в Роскомнадзоре. Пока доступно создание только одного такого канала. Вам потребуется [войти на платформу или зарегистрироваться](./02_Подключение_к_платформе/connection.md#этап-1-вход-на-платформу), [создать и верифицировать](./02_Подключение_к_платформе/connection.md#создание-организации) организацию.

После этого вы сможете добавить канал через бота [Каналы в MAX для бизнеса](https://max.ru/business_channel_bot). Название канала вы выбираете сами, а ник генерируется автоматически. Обратите внимание, что профиль в MAX должен быть создан на тот же номер телефона, что вы использовали при регистрации на платформе.

Для безопасности мы разделили добавление и повседневное использование канала между двумя платформами:
- В MAX для партнёров вы верифицируете организацию и инициируете создание канала
- В мессенджере MAX вы непосредственно создаёте канал, управляете его настройками и публикуете посты

[Создать канал с выбранным вами названием и автогенерируемым ником](../05_Каналы/create.md#как-создать-канал-с-выбранным-вами-названием)

### Приватные каналы

Приватный канал можно создать прямо в мессенджере MAX.

Для [создания приватного канала](../05_Каналы/create.md#приватные-каналы) подключаться к платформе MAX для партнёров не нужно — достаточно быть пользователем мессенджера MAX с аккаунтом, зарегистрированным на российский номер телефона. Рекомендуем использовать телефон организации или ответственного, кто будет управлять каналом и [модерировать заявки](../05_Каналы/manage.md#как-включить-или-отключить-заявки-на-вступление-в-канал--только-для-приватных-каналов) на вступление.

Для дополнительной безопасности вы можете [включить](../05_Каналы/manage.md#как-включить-или-отключить-заявки-на-вступление-в-канал--только-для-приватных-каналов) в приватных каналах опцию «Заявка на вступление»: по умолчанию она отключена. В этом случае пользователи смогут подписаться на канал только после согласования заявки администратором.

[Создать приватный канал](../05_Каналы/create.md#приватные-каналы)
