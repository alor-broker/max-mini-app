# Typography

Система типографики для MAX UI.

## Импорт

```jsx
import { Typography } from '@maxhub/max-ui';
```

## Подкомпоненты

- `Typography.Title` — заголовки
- `Typography.Headline` — подзаголовки
- `Typography.Body` — основной текст
- `Typography.Label` — метки и подписи
- `Typography.Action` — текст действий/кнопок
- `Typography.Display` — крупный дисплейный текст

## Использование

```jsx
<Typography.Title>Заголовок страницы</Typography.Title>
<Typography.Body>Основной текст сообщения</Typography.Body>
<Typography.Label>Подпись к полю</Typography.Label>
```
