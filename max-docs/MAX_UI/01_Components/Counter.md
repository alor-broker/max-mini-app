# Counter

Счётчик для отображения количества.

## Импорт

```jsx
import { Counter } from '@maxhub/max-ui';
```

## Использование

```jsx
<Counter value={5} />
<Counter value={99} max={99} />
```

## Свойства

| Свойство | Тип | Описание |
|----------|-----|----------|
| **value** | number | Текущее значение |
| **max** | number | Максимальное отображаемое значение (покажет "99+") |
| **mode** | `'primary' \| 'secondary'` | Режим отображения |
