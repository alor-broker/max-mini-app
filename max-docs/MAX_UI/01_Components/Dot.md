# Dot

Точка-индикатор (badge).

## Импорт

```jsx
import { Dot } from '@maxhub/max-ui';
```

## Использование

```jsx
<Dot />
<Dot color="red" />
```

## Свойства

| Свойство | Тип | Описание |
|----------|-----|----------|
| **color** | string | Цвет точки |
| **size** | `'small' \| 'medium' \| 'large'` | Размер точки |
